Introdução

O html é linguagem de marcação de hipertexto, é o componente básico para fazer o formato de uma página, exemplo: textos, formatos, parágrafos, navegação entre links e muito mais. Não configura-se como uma linguagem de programação, pois não tem um conjunto de instruções lógicas a serem seguidas.

Estrutura básica de um html

      <!DOCTYPE html>
      <html lang="en">
        <head>

          < meta charset="UTF-8">
            
          <meta http-equiv="X-UA-Compatible" content="IE=edge">
            
          <meta name="viewport" content="width=device-width, initial-scale=1.0">
          
          <title></title>
          
        </head>

        <body>

        </body>
      <html>
Todo inicio de um html tera esse formato. É o inicio do documento.

Principais tags do html:

Tags de títulos

  • <h1> - maior valor hierárquico
  • <h2>
  • <h3>
  • <h4>
  • <h5>
  • <h6> - menor valor hierárquico

Tags de Texto

  • <p> forma um parágrafo
  • <span> Usada muita das vezes como legendas
  • <pre> Usada para um texto pré-formatado
  • <b> formata o texto em negrito
  • <i> formata o texto em itálico
  • <br> quebra de linha

Tags de multimídia

  • <Img>
  • <video>
  • <Audio>

Tags de listas

  • <ul> lista não ordenadas
  • <ol> lista ordenadas
  • <li> linhas
Detalhe importante a tag <li> deve estar dentro de uma tag <ul> ou <ol>

Outra tag importante é a tag <a> usada para redirecionar, navegar entre links

Principais atributos

Os atributos abaixos servem para identificar determinada tags para fins de estilização, links ou até mesmo para serem usadas no JS.

  • Class
  • Id
  • Name
